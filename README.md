# Windows Event Viewer - SOC Analysis Project

## 📌 Project Overview

This project demonstrates how Windows Event Viewer logs can be collected, analyzed, and used for Security Operations Center (SOC) monitoring.

Windows Event Viewer provides detailed system activity records such as user logins, process activity, security changes, and system events. SOC analysts use these logs to detect suspicious behavior, investigate incidents, and perform threat analysis.

---

## 🎯 Project Objectives

- Understand Windows Event Logs
- Analyze security-related events
- Detect suspicious user activities
- Identify possible security incidents
- Practice SOC investigation techniques

---

## 🖥️ Environment

- Operating System: Windows 10 / Windows 11
- Tool: Windows Event Viewer
- Role: SOC Analyst Practice Lab

---

Used for:

- User authentication monitoring
- Account activity tracking
- Privilege changes
- Security auditing


Important Event IDs:

| Event ID | Description | SOC Use |
|---|---|---|
| 4624 | Successful Login | Track user access |
| 4625 | Failed Login | Detect brute force attempts |
| 4634 | Logoff | User session tracking |
| 4648 | Explicit Credential Login | Detect suspicious login |
| 4672 | Admin Privileges Assigned | Monitor privilege escalation |
| 4720 | User Account Created | Detect unauthorized accounts |
| 4726 | User Account Deleted | Account investigation |
